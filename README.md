# Motor-website

## Как начать?
Для того, чтобы начать работу, необходимо выполнить следующие команды
```
git clone <SSH address of this repo>
cd Motor-website/
python3 -m venv venv
source venv\bin\activate
pip install -r requirements.txt
```
